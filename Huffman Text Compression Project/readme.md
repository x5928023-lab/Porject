# Text Data Compression and Encoding Project

## Overview
This project implements a Huffman coding system for text compression and reconstruction.

It reads raw text data, calculates character frequency distributions, builds a Huffman tree using a priority queue, and generates prefix-free binary codes for encoding. The project also includes a decoding pipeline that reconstructs the original text from the compressed output using the saved tree structure.

## Files Included
- `encode.py` — builds the Huffman tree and encodes the input text
- `decode.py` — decodes the compressed bitstring back into text
- `sample_input.txt` — example input text used for testing
- `encoded.bin` — compressed output generated by the encoder
- `tree.pkl` — serialized Huffman tree used for decoding
- `reconstructed.txt` — decoded output for verification

## Key Features
- Character frequency analysis
- Huffman tree construction using a min-heap
- Prefix-free binary code generation
- End-to-end encoding and decoding pipeline
- Compression statistics reporting
- Output reconstruction verification
